# 📊 SenteInsights

Transform your mobile money statements into actionable financial insights. SenteInsights is a modern web application that analyzes transaction data from MTN MoMo, Airtel Money, and other mobile money providers, providing comprehensive financial analytics and visualizations.

## 🎯 Key Features

- **PDF Statement Parsing**: Upload and automatically parse mobile money statements (MTN MoMo, Airtel Money, etc.)
- **Transaction Analysis**: Detailed breakdown of income sources, spending patterns, and account activity
- **Financial Dashboards**: Interactive charts and visualizations for spending, income, and trends
- **Account Management**: Track multiple accounts with detailed account information and analytics
- **Data Conversion**: Convert and export transaction data in multiple formats
- **User Authentication**: Secure login and registration with Clerk authentication
- **API Usage Tracking**: Monitor your API usage and subscription limits
- **Document Processing**: Advanced PDF parsing with support for various statement formats
